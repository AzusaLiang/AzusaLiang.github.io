关于我如何在博客上放置LIVE2D的那些事

四月 18, 2020
  • web

相信有许多小伙伴和我一样，想在博客上放置可爱的live2d角色（嘘）
下面分享一下关于我如何放置LIVE2D以及在上面遇到的坑。
（1）首先打开blog的根目录下打开git bash

（2）输入我们的指令
npm install --save hexo-helper-live2d
这条命令帮助我们安装相对应的配置啊，环境什么的。

（3）然后呢我们在hexo的配置文件_config.yml下加入下面的配置

(4)大触也为我们提供了一些模型
$ npm install live2d-widget-model-wanko
(live2d-widget-model-wanko替换成你想要的模型)
模型到这里看吧 https://github.com/xiazeyu/live2d-widget-models

(5)我觉得肯定有部分同道之人喜欢别的live2d,像我就整了一个这样的[doge]

像这样的，我们首先百度一下，LIVE2D
在网上找找有哪些LIVE2D的资源，然后就是下载下来咯。
在blog的根目录下创建live2d_models的文件夹
然后将下载的文件解压到该文件夹中，
并且在blog的根目录配置文件中，对live2d配置中进行修改 ues: 模型文件名
这样我们就能使用我们独特的live2d模型了。

(6)有时我们会遇到我们的live2d模型放到了我们不想放的位置，挡住了一些内容。（这个我找了很久（哭））
这时我们打开 博客根目录下\node_modules\live2d-widget\src\config\defaultConfig
文件中的display中的hOffset与vOffset就是我们进行模型偏移的参数。
我们将hOffset与vOffset复制到 博客配置文件中，然后通过调整数值来调整位置,如
hOffset: 0
vOffset: 20

defaultConfig这个文件中还有很多参数可以拿来用。

来吧！！！拿起天下第一的LIVE2D去玩耍吧！！！
